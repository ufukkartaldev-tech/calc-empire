# CalcEmpire - Global Engineering Toolbox

CalcEmpire is a comprehensive 'Global Engineering Toolbox' designed to provide engineers, students, and hobbyists with a wide range of calculation tools and resources.

## Features
- Multi-language support (English, Turkish, Russian, Hindi, Japanese)
- Built with Next.js 15 (App Router)
- Styled with Tailwind CSS
- Tested with Vitest and React Testing Library

## Getting Started
1. Install dependencies: `npm install`
2. Run the development server: `npm run dev`
3. Run tests: `npm test`
